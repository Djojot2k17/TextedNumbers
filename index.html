Numbers

Numbers have three basic units: Ones, Tens, and Hundreds

Hundreds Tens Ones
100 = One Hundred 010 = Ten 001 = One
200 = Two Hundred 020 = Twenty 002 = Two
300 = Three Hundred 030 = Thirty 003 = Three
400 = Four Hundred 040 = Forty 004 = Four
500 = Five Hundred 050 = Fifty 005 = Five
600 = Six Hundred 060 = Sixty 006 = Six
700 = Seven Hundred 070 = Seventy 007 = Seven
800 = Eight Hundred 080 = Eighty 008 = Eight
900 = Nine Hundred 090 = Ninety 009 = Nine

The numbers 11,12,13,14,15,16,17,18,19 do not fit as part of the table above. If they did, 11 would read as: Ten One.

11: Eleven, 12: Twelve, 13: Thirteen, 14: Fourteen, 15: Fifteen, 16: Sixteen, 17: Seventeen, 18: Eighteen, 19: Nineteen

Commas are used if the number is greater than 999. Example: 1,000

If reading from right to left, the first comma = thousand, the second comma = million, the third comma = billion etc...
